# Logging

Для логирования необходимо использовать **SLF4J + Logback**, придерживаясь следующей логики выбора уровня:

- **ERROR** — критические ошибки. Только то, что требует немедленного внимания (упал внешний сервис, NPE в продакшне).
- **WARN** — нестандартные ситуации, но сервис работает («Пользователь ID=123 ввёл неверный пароль 5 раз подряд» или response time от внешней системы превысил 5 секунд).
- **INFO** — ключевые бизнес-события (создано отключение, внешняя система зарегистрирована).
- **DEBUG** — для локальной отладки, в продакшне выключен.
